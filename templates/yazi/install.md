## Build from source
https://yazi-rs.github.io/docs/installation/#build-from-source

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
git clone https://github.com/sxyazi/yazi.git && cd yazi && cargo build --release --locked
./target/release/yazi
```
